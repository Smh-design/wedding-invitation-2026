# 孙鸣航 & 顾紫薇婚礼邀请函H5长页

## 项目概述
这是一个专为孙鸣航和顾紫薇设计的交互式婚礼邀请函H5长页，完全遵循用户提供的视觉设计规范，具有温柔、治愈、浪漫、极简的风格特点。

## 设计特色
- **视觉风格**：日系写真书的高级排版质感，微弱的哑光艺术纸纹理和电影般的胶片质感
- **色彩系统**：高级灰绿(#8A9A5B) + 纯白(#FFFFFF) + 低饱和度经典红(#C62828)
- **字体层级**：手写体(主视觉) + 经典衬线体(英文) + 干净无衬线体(说明文字)
- **光影设定**：Golden Hour（黄金时刻）的自然柔和漫反射光

## 项目结构
```
wedding-invitation/
```

## 功能模块

### 1. 主视觉封面 (Cover Module)
- 顶部英文诗句 + 音乐图标
- 中心中文手写体"好久不见，婚礼见～"
- 核心双人合照 (AL5I5723.JPG)
- 新人姓名拼音+中文显示
- 婚礼日期和地点信息
- H5导航栏 (祝福留言、烟火贺喜、永结同心)

### 2. 我们的故事 (Love Story Module)
- 过渡文本："合情 | 合理 | 合法 | 和你"
- 英文标题"OUR LOVE STORY" + 红色标签
- 故事照片 (AL5I5908.jpg) + "Sweet"标签
- 英文诗句 + 中文爱情故事描述

### 3. 时间与空间 (Time & Location Module)
- 倒计时设计 (天/时/分/秒实时更新)
- 双人照片墙 (AL5I2237-1.jpg, AL5I2302.JPG)
- 地图导航功能 (可跳转Google Maps)
- 酒店详细信息

### 4. 宾客回执 (RSVP Module)
- 表单设计：姓名 + 出席人数输入
- 红色边框输入框 + 红星提示符
- 深红色提交按钮
- 结尾祝福语："敬备喜酌"、"恭候莅临"
- 底部固定导航栏

## 交互功能
- **实时倒计时**：自动计算距离2025年5月4日的剩余时间
- **表单验证**：客户端验证 + 成功提交反馈
- **导航交互**：
  - 祝福留言：弹出祝福语输入框
  - 烟火贺喜：屏幕烟花特效
  - 永结同心：爱心飘动画
- **平滑滚动**：页面间平滑滚动 + 导航高亮
- **图片懒加载**：滚动时淡入效果
- **响应式设计**：适配桌面、平板、手机

## 技术特点
- **纯前端技术**：HTML5 + CSS3 + JavaScript
- **现代CSS特性**：CSS变量、Flexbox、Grid、CSS动画
- **响应式设计**：移动端优先，多断点适配
- **性能优化**：图片懒加载、CSS硬件加速
- **字体系统**：Google Fonts (Cormorant Garamond + Inter)
